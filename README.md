# -LETHALITY-CALCULATOR

# :green_circle: DELTA GREEN // LETHALITY PROTOCOL CALCULATOR
クトゥルフ神話TRPG（7版）環境にデルタグリーンの「致死率（Lethality）」ルールを導入・自動計算・判定シミュレートするためのウェブツールです。

## :bar_chart: 基本仕様（入力パラメータ）
* **WEAPON_DAMAGE**: 武器の最大ダメージ（例: 2d8なら「16」）。
* **VOLLEY_COUNT**: 発射弾数。
* **TARGET_ARMOR**: ターゲットの装甲値。
  * **特殊入力対応**: `3*2,1*4`（装甲3が2体、装甲1が4体）のように入力すると、自動で対象グループの平均装甲値を算出して適用します。
* **TARGET_COUNT**: ターゲット数。
  * 巻き込み上限はシステム上「弾数÷3」体に自動制限されます。

## :gear: 計算ロジックと補正機能
* **乱射の面制圧補正**: 複数対象の場合、弾幕の分散効率を計算します。単体よりも2〜3人の密集地帯へ弾幕を浴びせることで、最も高い総致死率（ダメージ効率）を発揮する仕様です。
* **神話生物の特殊耐性**: 物理装甲の減算前に適用されるダメージ軽減処理です。「半減」「1/4」「弾丸耐性（入力ダメージ1固定）」を選択可能です。
* **装甲による減衰・無効化**:
  * 武器ダメージが装甲を上回れない場合、致死率は強制的に「0%（無効）」となります。
  * ダメージが貫通した場合、計算式上における装甲の減衰力は最大「5」でキャップされ、重装甲であっても一定の致死率を担保します。

## :game_die: 判定とダメージ処理（CoC7版準拠）
実行ボタンを押すと1d100がシミュレートされ、以下の基準で結果を出力します。命中判定の出目をそのまま流用する「一括解決型」の処理にも対応可能です。

* **[致死判定 成功] (出目 ≦ 致死率)**
  * 対象は即死せず、即座に **HP0＋重傷（＝瀕死状態）** へ強制移行します。次ラウンド終了時までにカバーリングや医学による救命措置がなければ死亡（ロスト）として処理されます。
* **[致死判定 失敗] (出目 > 致死率)**
  * 「出目の10の位＋1の位（出目100は0）」から装甲値を引いた値が、通常ダメージとしてHPから減算されます。
* **[無効] (致死率 0%)**
  * 装甲または特殊耐性により、攻撃が完全に阻まれた状態として処理されます。
# :green_circle: DELTA GREEN 致死率システム // CoC7版 導入・運用ガイドライン

## :pushpin: 主旨
デルタグリーン（DG）の「致死率（Lethality）」ルールをクトゥルフ神話TRPG（7版）に輸入しつつ、完全なコピーではなく、CoC特有の多様な武器データや戦術状況に適応できるよう再構築した独自のダメージ処理システムです。

## :dart: 使用用途
主に**連射（7版におけるボレー / フルオート射撃）の処理**を劇的に軽量化・戦術化するために使用します。3発発射から3点バースト、マガジン全弾を撃ち尽くすフルバーストまで、あらゆる連射状況のダメージ算出を1つのシステムで統合管理します。

## :gear: 運用方針

### 1. 致死＝瀕死への移行（ステータス処理）
本システムにおいて、致死判定（Lethalityロール）に成功した場合、いかなる最大HPのキャラクターであっても、**即座にルールブック準拠の「瀕死」状態へ移行**します。これによりダイス1発での「即死（即時ロスト）」を防ぎつつ、次ラウンド終了時までの「医学や応急手当による救命の猶予」という劇的な戦術的ジレンマを生み出します。

### 2. 射撃・ダメージ判定の処理
複数対象や複数射撃を行う場合でも、煩雑なペナルティ計算は行わず、**1度の命中判定（射撃技能ロール）**を行い、その後ダメージロールの代わりに致死率ロールを行います。
* **対象への処理:** 原則として、巻き込んだ対象（個体）ごとに致死率ロールを行って個別にダメージを算出しますが、ゲームのテンポを最優先する場合は、1回の致死率ロール結果を全員に一括適用して簡略化することも可能です。

### 3. 固定値としての運用（事前計算）
セッション中に毎回計算ツールを使用するのではなく、事前に武器ごとの「固定致死率」を割り出してキャラクターシートに記入しておく運用を推奨します。
* **算出基準:** 「装甲2」「対象1体」「その武器の標準的なフルバースト連射数」を入力して算出された値を、その武器の基本致死率とします。
* **探索者の素人設定:** デルタグリーンのような訓練された専門部隊ではなく、一般的な探索者（素人）が乱射する状況を表現したい場合は、計算機に入力する「連射数」を半分に減らすことで、命中精度の低さを致死率の低下としてシステム的に表現できます。

## :warning: キーパー（KP）への推奨事項

### プレイヤー（探索者）への「装甲」によるKPバフ方針
本システムは非常に殺傷能力が高いため、KPが探索者の生存率をコントロールするための調整（バフ）として「装甲」を活用することを推奨します。

* **[推奨] 探索者への装甲1〜2の付与（防弾チョッキ等）**
  探索者に「1」または「2」の装甲（防弾チョッキや厚手の防具など）をアイテムとして与えることは、非常に効果的な生存バフとなります。本システムの計算式上、低い装甲値であっても致死率を適度に抑え込む効果が働くため、「撃たれれば致命傷になり得るが、即座の全滅は避けやすくなる」という適切な緊張感と生存の余地をプレイヤーに提供できます。
* **[非推奨] 探索者への装甲5以上の付与**
  装甲値が5を超えると、計算式の分母キャップ上限に達し、敵の一般的な銃器による致死率が極端に（あるいは完全に）無効化されやすくなります。タクティカル・ホラーとしての銃撃戦の緊張感が消失してしまうため、過剰なバフには注意してください。
